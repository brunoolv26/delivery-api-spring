# Delivery API - Projeto de Portfólio

Este projeto é uma API REST funcional para gerenciamento de cardápio de delivery, desenvolvida para colocar em prática conceitos de Spring Boot e Persistência de Dados.

## Sobre o Projeto
A ideia surgiu da minha vivência diária trabalhando com entregas. Uni meu conhecimento de campo como motoboy com os estudos de Engenharia de Software para criar uma ferramenta que resolve problemas reais de uma pizzaria.

## Tecnologias Utilizadas
- Java 21
- Spring Boot 4.0.6
- Spring Data JPA
- H2 Database

## Funcionalidades
- GET /produtos: Lista o cardápio.
- POST /produtos: Cadastra novos itens.
- DELETE /produtos/{id}: Remove produtos pelo ID.
